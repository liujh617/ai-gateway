# v0.1.4 发布计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 发布 v0.1.4，包含单模型元数据检索、Response 读取与删除、store/provider-health 锁优化。

**Tech Stack:** Go 1.22 standard library; WSL `Ubuntu-24.04` 执行所有验证。

## Global Constraints

- Prefer the Go standard library and add no dependency.
- 遵循 SemVer，版本号 0.1.4。
- 发布前通过 `make release-check VERSION=0.1.4`。
- Changelog 从 Unreleased 移动到 `## 0.1.4 - 2026-07-16`。
- 创建 annotated tag `v0.1.4` 并推送。
- 不在发布中暴露任何真实 API key。
